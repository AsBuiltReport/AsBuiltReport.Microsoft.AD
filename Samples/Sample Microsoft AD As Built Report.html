1.1.5.5 Inter-Site Transports

Site links in Active Directory represent the inter-site connectivity and method used to transfer replication traffic. There are two transport protocols that can be used for replication via site links. The default protocol used in site link is IP, and it performs synchronous replication between available domain controllers. The SMTP method can be used when the link between sites is not reliable.

Best Practice: The infrastructure master role in the domain ACAD.PHARMAX.LOCAL should be held by a domain controller that is not a global catalog server.
